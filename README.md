# Clinic Management System

A complete clinic and medical office management system built with Django, designed to streamline patient management, appointments, medical records, and financial operations.

This project was developed as a real-world portfolio application focused on backend architecture, authentication, scheduling systems, and healthcare workflow management.

---

# 🚀 Features

## 👨‍⚕️ Medical Management
- Doctor registration and management
- Specialty organization
- Doctor schedules and availability

## 🧑‍🤝‍🧑 Patient Management
- Patient registration
- Contact and profile management
- Patient history tracking

## 📅 Appointment Scheduling
- Create, edit, and cancel appointments
- Daily and monthly agenda
- Appointment status management

## 📄 Medical Records
- Patient medical history
- Consultation notes
- Clinical record organization

## 💰 Financial Module
- Payment tracking
- Financial control dashboard
- Revenue management

## 🔐 Authentication & Security
- User authentication system
- Access control by users
- Django Admin support

## 📊 Administrative Features
- Dashboard and management interface
- Organized database structure
- Responsive system interface
